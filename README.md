# 🌐 Personal 3D Portfolio Website

An interactive, visually engaging personal portfolio built with **React**, **Tailwind CSS**, and **Three.js** — showcasing animation, 3D rendering, and responsive web design.

---

## 🚀 Tech Stack

**Frontend:**
- React.js  
- Tailwind CSS  
- Framer Motion  
- Typed.js  

**3D & Animation:**
- Three.js  
- @react-three/fiber  
- @react-three/drei   

**Version Control:**
- Git  
- GitHub  

---

## 💫 Portfolio Highlights

### 🎨 Interactive 3D Portfolio Website
A personal portfolio showcasing 3D elements, smooth animations, and a responsive layout.

**Key Implementations:**
- Built with **React**, **Tailwind CSS**, and **Framer Motion** for dynamic transitions  
- Integrated **Three.js** (*react-three/fiber* & *drei*) for interactive 3D visuals  
- Added **Typed.js** for animated typing effects  
- Fully responsive and optimized for desktop & mobile  

🔗 **Live Demo:** [https://nakulkumarportfolio.netlify.app/](https://nakulkumarportfolio.netlify.app/)
